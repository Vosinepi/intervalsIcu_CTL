Un insipiente proyecto para mostrar en ventana los datos de entrenamiento ciclisto traidos de IntervalsIcu mediante el uso de su Api.

IntervalsIcu https://intervals.icu/

API https://forum.intervals.icu/t/api-access-to-intervals-icu/609

Se renderiza mediante TKinder en una ventana semitransparente. 


En los ajsutes de usuario en la parte de desarrollador se necesita crear la API_KEY para poder acceder. 

api_token = "reemplazar por API_KEY"
id_atleta = "id_atleta"
